## Цель работы
Изучение структурного паттерна проектирования «Декоратор» на Python. 
Реализация системы получения курсов валют с API Центрального Банка РФ 
с возможностью конвертации данных в форматы CSV и YAML и сохранения в файл
соответствующего формата.
## Теоретическая часть
1. Декоратор — структурный шаблон, позволяющий динамически добавлять объектам новую 
функциональность через обёртку. В отличие от наследования, декорирование возможно во время 
выполнения программы.
2. Абстрактный базовый класс (ABC) — механизм Python (`abc.ABC`, `@abstractmethod`) 
для определения обязательного интерфейса. Гарантирует, что все конкретные реализации 
предоставят требуемые методы.
## Выполнение работы
### Базовый компонент: получение данных с API ЦБ
Реализован конкретный компонент `CurrenciesCBR`, отвечающий за взаимодействие с внешним сервисом.
Класс `CurrenciesCBR` наследуется от абстрактного `Currencies` и реализует:
1. `get_currencies() -> Dict[str, Any]:` выполняет HTTP-запрос, валидирует наличие ключа "Valute", возвращает словарь валют.
2. `save_to_file(filepath: str) -> None:` сохраняет сырые данные в JSON-файл с кодировкой UTF-8.
### Декоратор для конвертации в CSV
Реализован декоратор `ConvertToCSV`, добавляющий функциональность экспорта в табличный формат.
Класс оборачивает объект `Currencies` и предоставляет:
1. `get_currencies_csv() -> List[Dict[str, Any]]:` преобразует вложенный словарь в список плоских словарей, добавляя поле val_id.
2. `save_to_file(filepath: str) -> None:` записывает данные в CSV с заголовками через `csv.DictWriter`.
3. Базовый метод `get_currencies()` переопределён так, чтобы возвращать `dict` (сохранение интерфейса), а конвертация вынесена в отдельный метод.
### Декоратор для конвертации в YAML
Реализован декоратор `ConvertToYAML` для экспорта в человекочитаемый формат.
Класс предоставляет:
1. `get_currencies() -> Dict[str, Any]:` делегирует вызов обёрнутому объекту (интерфейс не нарушен).
2. `get_currencies_yaml() -> str:` возвращает YAML-представление данных как строку через `yaml.dump()`.
3. `save_to_file(filepath: str) -> None:` записывает YAML-строку в файл.
### Интерфейс через ABC
Реализован абстрактный класс Currencies, задающий контракт для всех компонентов.
```python
class Currencies(ABC):
    @abstractmethod
    def get_currencies(self) -> Dict[str, Any]: ...
    @abstractmethod
    def save_to_file(self, filepath: str) -> None: ...
```
